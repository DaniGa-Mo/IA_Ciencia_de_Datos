# IA_Ciencia_de_datos
En este módulo aprenderás a usar ChatGPT como asistente para optimizar el análisis y visualización de datos. Aprenderás a diseñar gráficos personalizados, conocerás las posibilidades y limitaciones de la IA y, finalmente, crearás un informe interactivo integrando ChatGPT con la herramienta Gamma.
